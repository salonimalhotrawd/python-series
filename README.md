## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/salonimalhotrawd/python-series.git
```

### Navigate to the project

```bash
cd python-series
```

### Run a Python file

```bash
python filename.py
```

or

```bash
python3 filename.py
```
